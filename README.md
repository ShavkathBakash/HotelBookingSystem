# 🏨 Hotel Booking System

A Java-based console application developed using Core Java, JDBC, and MySQL for managing hotel information and customer bookings. The project demonstrates CRUD operations, database connectivity, Object-Oriented Programming, and exception handling.

---

## 📌 Features

### Hotel Management

- Add Hotel
- View Hotels
- Search Hotel
- Update Hotel
- Delete Hotel

### Booking Management

- Add Booking
- View Bookings
- Search Booking
- Cancel Booking

---

## 🛠 Technologies Used

- Core Java
- JDBC
- MySQL
- VS Code
- MySQL Connector/J

## 🚀 How to Run

1. Install Java (JDK 17 or above)
2. Install MySQL
3. Create the database using `database.sql`
4. Add MySQL Connector JAR inside the `lib` folder.
5. Open the project in VS Code.
6. Run `Main.java`.
